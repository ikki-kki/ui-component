
# 페이지 네이션 구현하기


## 프로젝트 목표
임시의 댓글 데이터가 있습니다.
아래와 같이 페이지를 클릭하면 각 페이지에 해당 하는 댓글을 불러오세요.

![페이지네이션](https://user-images.githubusercontent.com/12206933/103061321-5d9c7600-45ee-11eb-99dc-1944a5981740.gif)



## 구현시 요청 사항
- 한 페이지당 보여질 게시물 수 : 5
- 첫 페이지 로딩시 게시물 보임
- 숫자 클릭시 해당하는 게시물 보임 + 각 페이지 위치 활성화
